About


COVID-19-Watch is a public online tool to monitor COVID-19's progression in the United States.

The data is updated daily so that anyone can share and bookmark COVID-19 Status Reports of interest (local, state, or national) with their community.


Key Features:

  • Find and share - Quickly find and share local COVID-19 Status Reports for anywhere in the nation
  • Local, state, and national - Explore data and trends at local, state, and national levels
  • Change over time - Watch change over time in the map and graph
  • Up-to-date data - Data is updated daily from Johns Hopkins

Access:

  • www.covid-19-watch.com
  • covid-19-watch.herokuapp.com - Alternative, sometimes better for firewalls

Up-to-Date Data

All information is calculated daily from the COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University. Read more about JHU CSSE's data process in their README. This Johns Hopkins data service is highly regarded as one of the best sources of COVID-19 data.

Data lifecycle:

  1. Raw COVID-19 data are automatically pulled from Johns Hopkins every morning around 3 a.m. by Diego the Data Broker, a web robot developed to support COVID-19-Watch.
  2. Diego calculates metrics such as case rate and acceleration for every county, state, and the nation for every week since the data begins
  3. This data is added to a Data Package alongside county, state, and national GeoJSON map files and added to the database
  4. When someone visits COVID-19-Watch online, Diego takes the morning's latest Data Package and gives it to the app for display in the map and Status Reports

Key Technologies and Services

The following technologies and services were instrumental in the development of COVID-19-Watch. This is the "tech stack".

  • COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University - Live COVID-19 data
  • Esri Living Atlas - Population numbers and county, state, and national map data
  • OpenStreetMap - Location search (geocoding service)
  • Leaflet - Dynamic web map
  • Bootstrap, Font Awesome - User interface components
  • Angular - JavaScript framework
  • Node, Express - web server
  • PostgreSQL - Database
  • Heroku - Development platform, deployment, and hosting
